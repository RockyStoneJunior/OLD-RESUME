Phodal's Resume
=================================================

> 这是在毕业时用Latex写的一个简单。

 - Windows 上可以用 MiKTex
 - Mac OS 上可以用 MacTex
 - GNU/Linux 上可以用 Texlive Pro

如需帮助: 请在这里 [https://github.com/phodal/RESUME/issues](https://github.com/phodal/RESUME/issues) 创建一个新的``issue``。

##简介

一共分为三个版本，大致是相同的。

 - Personal目录下是``web``和``嵌入式``
 - Embedded目录下是``嵌入式``专用
 - WebDevel目录下是``Web``专用
 - En目录下是英文简历
